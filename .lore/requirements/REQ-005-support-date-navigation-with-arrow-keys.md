# REQ-00X - Support date navigation with arrow keys

## Requirement

The user shall be able to navigate between dates using the keyboard.

## Acceptance Criteria

- Pressing Left Arrow selects the previous date.
- Pressing Right Arrow selects the next date.
- Date navigation updates the dashboard immediately.
