# REQ-00Z - Preserve selected date while navigating

## Requirement

The dashboard shall maintain the currently selected date until the user changes it.

## Acceptance Criteria

- The selected date is visible in the interface.
- The dashboard does not automatically jump back to the current date.
- Navigation remains possible in both directions.
