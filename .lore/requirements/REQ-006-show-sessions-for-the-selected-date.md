# REQ-00Y - Show sessions for the selected date

## Requirement

The dashboard shall display sessions that were active on the currently selected date.

## Acceptance Criteria

- [ ] The selected date determines which sessions are shown.
- [ ] Sessions are not limited to the last 24 hours.
- [ ] Changing the selected date refreshes the session list.
